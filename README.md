# For Jinji

A simple static love-letter website for Jinji.

## Local Preview

Open `index.html` in a browser, or run a tiny local server:

```bash
python3 -m http.server 3001
```

Then visit `http://localhost:3001`.

## Publish With GitHub And Vercel

1. Create a new GitHub repository.
2. Push this folder to the repository.
3. In Vercel, choose **Add New Project** and import that GitHub repository.
4. Keep the framework preset as **Other** or **Static**.
5. Leave build settings empty because this site is plain HTML/CSS.
6. Deploy.
